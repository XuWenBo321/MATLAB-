## 元胞数组
  以其他数组的副本为元素的多维数组。
```bash
  创建方式：1.使用cell函数，创建空矩阵的元胞数组
           2.将其他内容的集合括如花括号{}
   例：C = {A sum(A) prod(prod(A))} //生成1*3元胞数组
        C = 
        [4x4 double]    [1x4 double]    [20922789888000]
    牢记：
    * 检索元胞中的内容，在花括号中使用下标，C{1}
    * 元胞数组包含其他数组的副本，但不包含指向这些数组的指针，即不会改变原矩阵的值
    
    M = cell(8,1); //生成8*1的元胞数组
    for n = 1:8  //for循环，对M(n)赋值，
       M{n} = magic(n); 
    end
    M
```
