## 矩阵
```bash
  #### 创建矩阵
    1)A = pascal(n)
    2)B = magic(3)
    3)C = randi(10,3,2) // 创建范围在10以内，3行2列的矩阵
  #### 矩阵四则运算
    1) 减法要求两个矩阵具有兼容的维度。
    2) 相乘，要求要求乘号前的矩阵列数等于乘号后的矩阵行数（m*n--n*k = m*k)
    >> u = [3;1;4];
    >> v = [2 0 -1];
    >> x = u*v

    x =

         6     0    -3
         2     0    -1
         8     0    -4

    >> x = v*u

    x =

         2
    3) 内积，标量积和点积
    4) 转置
      对于复数向量或者矩阵z，转置可将其中的每个附属元素转化为其复共轭数。
      z = [1+2i 7-3i 3+4i; 6-2i 9i 4+7i]
      z =

         1.0000 + 2.0000i   7.0000 - 3.0000i   3.0000 + 4.0000i
         6.0000 - 2.0000i   0.0000 + 9.0000i   4.0000 + 7.0000i
      z 的复共轭转置为：

      z'
      ans =

         1.0000 - 2.0000i   6.0000 + 2.0000i
         7.0000 + 3.0000i   0.0000 - 9.0000i
         3.0000 - 4.0000i   4.0000 - 7.0000i
       ** 复数矩阵正常转置，使用z.'
    5) 乘法逻辑
      >> for i = 1:m
           for j = 1:n
             C(i,j) = A(i,:)*B(:,j);
           end
      end
      >> C

      C =

          15    15    15
          26    38    26
          41    70    39

    **乘法不具有交换律
  #### 单位矩阵
     eye(m,n) //返回m*n的单位矩阵
     eye(n) //返回n*n的单位矩阵
```
